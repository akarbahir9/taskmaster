# 🔧 Cursor IDE App Login Issues - Troubleshooting Guide

## 🚨 Common Cursor IDE Login Problems & Solutions

### **Problem 1: "Can't Sign In" in Cursor App**

**Solutions:**
1. **Close Cursor completely** and restart
2. **Clear Cursor cache:**
   - Windows: `%APPDATA%\Cursor`
   - Mac: `~/Library/Application Support/Cursor`
   - Linux: `~/.config/Cursor`
3. **Log out and log back in:**
   - Go to Cursor → Settings → Account
   - Click "Sign Out" then "Sign In"

### **Problem 2: "Network Connection Error"**

**Solutions:**
1. **Check internet connection**
2. **Disable VPN** temporarily
3. **Check firewall settings**
4. **Try different network** (mobile hotspot)

### **Problem 3: "Account Authentication Failed"**

**Solutions:**
1. **Reset password** on cursor.sh website
2. **Use browser login:**
   - In Cursor app → Help → "Sign in with Browser"
3. **Clear browser cookies** for cursor.sh
4. **Try incognito/private browser mode**

### **Problem 4: "App Stuck on Loading"**

**Solutions:**
1. **Force quit Cursor** and restart
2. **Run as administrator** (Windows)
3. **Update Cursor** to latest version
4. **Reinstall Cursor:**
   - Uninstall current version
   - Download fresh from cursor.sh
   - Install and try login

### **Problem 5: "Two-Factor Authentication Issues"**

**Solutions:**
1. **Check 2FA app time sync**
2. **Use backup codes** if available
3. **Disable 2FA temporarily** on website
4. **Contact Cursor support**

## 🆘 Emergency Solutions

### **Nuclear Option - Complete Reset:**
1. **Uninstall Cursor completely**
2. **Delete all Cursor folders:**
   - Windows: `%APPDATA%\Cursor` and `%LOCALAPPDATA%\Cursor`
   - Mac: `~/Library/Application Support/Cursor`
   - Linux: `~/.config/Cursor`
3. **Restart computer**
4. **Download fresh Cursor** from cursor.sh
5. **Install and login**

### **Alternative - Use Browser Version:**
1. **Go to** cursor.sh
2. **Use web-based editor** if app won't work
3. **Contact support** for app issues
