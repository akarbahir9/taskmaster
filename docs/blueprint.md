# **App Name**: TaskMaster

## Core Features:

- Add Task: Add a new task to the todo list.
- Complete Task: Mark a task as complete.
- Delete Task: Delete a task from the todo list.
- Task Display: Display all tasks in a clear, organized list.
- Task Filter: Filter tasks to show only active or only completed.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) to promote focus and organization.
- Background color: Light gray (#F0F4F8) for a clean, uncluttered feel.
- Accent color: Soft green (#81C784) for completed tasks.
- Body and headline font: 'PT Sans', a humanist sans-serif, providing both modern look and warmth for readability.
- Simple, clear icons for task actions (add, complete, delete).
- Clean, intuitive layout with a clear visual hierarchy.
